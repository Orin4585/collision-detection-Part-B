# 2D Collision Detection Simulation

## Prerequisites

1. **DownloadVisual Studio or MinGW**

2. **Download CMake**

3. **Download StarUML**

## Building the Project

1. Create a build directory:
```bash
mkdir build
cd build
```

2. Generate build files:
```bash
cmake ..
```

3. Build the project:
```bash
cmake --build .
```

## Project Structure
- `src/` - Source files
- `include/` - Header files
- `docs/` - Documentation and UML diagrams
- `tests/` - Test files
